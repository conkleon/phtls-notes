# Εκτίμηση Καταπληξίας

**Περίληψη**: Συστηματική αξιολόγηση για καταπληξία με τα πλαίσια XABCDE και MARCH, με έμφαση στα πρώιμα σημεία (ταχύπνοια, ταχυκαρδία, αλλαγές δέρματος) έναντι των όψιμων (υπόταση), και επίγνωση παραγόντων που μπορούν να αποκρύψουν τη σοβαρότητα.

**Πηγές**: PHTLS Prehospital Trauma Life Support 10th Edition.pdf

**Τελευταία ενημέρωση**: 2026-04-14

---

## Πλαίσια Πρωτογενούς Εκτίμησης

### XABCDE (Τυπική Τραυματολογική ΕΜΥ)

| Βήμα | Εστίαση |
|------|---------|
| **X** — εΞωτερική αιμορραγία | Άμεσος έλεγχος απειλητικής για τη ζωή εξωτερικής αιμορραγίας |
| **A** — Αεραγωγός (Airway) | Βατότητα με έλεγχο αυχενικής σπονδυλικής κατά περίπτωση |
| **B** — Αναπνοή (Breathing) | Αερισμός και οξυγόνωση |
| **C** — Κυκλοφορία (Circulation) | Αιμάτωση και πρόσθετη αιμορραγία |
| **D** — Αναπηρία (Disability) | Νευρολογική κατάσταση / διαταραχή συνείδησης |
| **E** — Έκθεση/Περιβάλλον (Expose/Environment) | Εντοπισμός κρυφής αιμορραγίας, πρόληψη υποθερμίας |

### MARCH (Τακτική/Στρατιωτική ΕΜΥ)

| Βήμα | Εστίαση |
|------|---------|
| **M** — Μαζική αιμορραγία | Τουρνικέ, πυελικός επίδεσμος, αιμοστατικός επίδεσμος |
| **A** — Αεραγωγός | Απόφραξη → θέση, ρινοφαρυγγικός αεραγωγός, προχωρημένος αεραγωγός |
| **R** — Αναπνοή | Τραύματα θώρακα, υπό τάση πνευμοθώρακας |
| **C** — Κυκλοφορία | Εκτίμηση καταπληξίας, IV/IO πρόσβαση, χορήγηση υγρών |
| **H** — Κεφαλή/Υποθερμία | Πρόληψη δευτερογενούς ΚΕΚ, διατήρηση θερμοκρασίας |

Και τα δύο πλαίσια δίνουν **προτεραιότητα στον έλεγχο αιμορραγίας πριν τον αεραγωγό** — αντιστροφή της παραδοσιακής αλφαβητικής σειράς ABC. (πηγή: PHTLS Prehospital Trauma Life Support 10th Edition.pdf)

## Πρωτογενής Εκτίμηση: Τι Αξιολογούμε

### Εξωτερική Αιμορραγία
- **Αξιολογείται πρώτη** — ένας ασθενής μπορεί να πεθάνει σε λεπτά από μείζον αρτηριακό τραύμα
- Έλεγχος για: λαχνιές τριχωτής κεφαλής, τραύματα επί μεγάλων αγγείων (υποκλείδιο, μασχαλιαίο, βραχιόνιο, κερκιδικό, ωλένιο, καρωτίδα, μηριαίο, ιγνυακό)
- Εάν δεν υπάρχει εξωτερική αιμορραγία → υποψία εσωτερικής έως απόδειξης του εναντίου

### Αεραγωγός
Ασθενείς που χρειάζονται άμεση διαχείριση αεραγωγού (κατά σειρά προτεραιότητας):
1. Δεν αναπνέουν
2. Εμφανής συμβιβασμός αεραγωγού
3. Θορυβώδης αναπνοή (στριδόρ, γουργούρισμα)
4. Έντονα παθολογικός ρυθμός αναπνοής

### Αναπνοή — Πρωιμότερο Σημείο Καταπληξίας
Αναερόβιος μεταβολισμός → γαλακτικό οξύ → σύστημα ρυθμιστικών → ↑ CO₂ → διέγερση αναπνευστικού κέντρου → **ταχύπνοια**.

> **Η ταχύπνοια είναι συχνά το πρωιμότερο σημείο καταπληξίας** — εμφανίζεται νωρίτερα από την ταχυκαρδία.

- Στην πρωτογενή εκτίμηση: εκτίμηση αναπνοών ως αργές / φυσιολογικές / γρήγορες / πολύ γρήγορες — όχι ακριβής μέτρηση
- **Αργές + καταπληξία** = βαθιά καταπληξία, πιθανή καρδιακή ανακοπή επικείμενη
- SpO₂ < 94% → ανησυχητικό, αναζητήστε αιτία
- ETCO₂ αναξιόπιστο για καθοδήγηση αερισμού σε ασθενείς με καταπληξία (κακή συσχέτιση με PaCO₂)
- Ασθενής που αφαιρεί μάσκα οξυγόνου + ανησυχία + σύγχυση = **εγκεφαλική ισχαιμία** («αίσθηση ασφυξίας»)

### Κυκλοφορία
Αξιολόγηση:
1. **Αιμορραγία** — ποσότητα και πηγή
2. **Αιμάτωση** — ολικού σώματος και περιφερική

Απουσία σφυγμού στην κερκιδική αρτηρία = σοβαρή υποογκαιμία. «Αδύναμος» κερκιδικός σφυγμός συσχετίζεται με ΑΠ ~26 mm Hg χαμηλότερη από «φυσιολογικό» και **15× υψηλότερη θνητότητα**.

### Αναπηρία (Διαταραχή Συνείδησης)
Διαταραχή συνείδησης = μειωμένη οξυγόνωση εγκεφάλου έως απόδειξης του εναντίου.

Έξι αιτίες διαταραχής συνείδησης στο τραύμα:
1. Υποξαιμία
2. Αγγειακό εγκεφαλικό επεισόδιο
3. Καταπληξία (μειωμένη εγκεφαλική αιμάτωση)
4. ΚΕΚ (Κρανιοεγκεφαλική Κάκωση)
5. Δηλητηρίαση (αλκοόλ, ναρκωτικά, δηλητήρια)
6. Μεταβολικά (διαβήτης, σπασμοί, εκλαμψία)

> **Αντιμετωπίστε πρώτα την υποξαιμία** — είναι η πιο θεραπεύσιμη και η πιο γρήγορα θανατηφόρα.

## Δευτερογενής Εκτίμηση: Ζωτικά Σημεία Αναλυτικά

### Αναπνευστικός Ρυθμός (Φυσιολογικό: 12–20 αναπν./λεπτό)
| Ρυθμός | Ερμηνεία |
|--------|----------|
| 20–30 αναπν./λεπτό | Οριακά παθολογικό — έναρξη καταπληξίας, συμπληρωματικό O₂ |
| > 30 αναπν./λεπτό | Όψιμη καταπληξία — απαιτείται υποβοηθούμενος αερισμός |

### Σφυγμός (Φυσιολογικό: 60–100 σφ/λεπτό)
| Ρυθμός | Ερμηνεία |
|--------|----------|
| 100–120 σφ/λεπτό | Πρώιμη καταπληξία |
| > 120 σφ/λεπτό | Σαφής καταπληξία (εκτός πόνου/φόβου) |
| > 140 σφ/λεπτό | Κρίσιμο |

### Αρτηριακή Πίεση — Όψιμο και Αναξιόπιστο Πρώιμο Σημείο
- **Δεν πέφτει μέχρι απώλεια ≥ 30% αίματος** (Τάξη III)
- Πτώση ΑΠ = ο ασθενής δεν μπορεί πλέον να αντιρροπήσει = **όψιμο, επικίνδυνο σημείο**
- Οι βλάβες εγκεφάλου δεν προκαλούν υπόταση έως εγκεφαλική κήλη — υπόταση + κεφαλική κάκωση = υπόθεση αιμορραγίας από άλλες κακώσεις
- **Εξαίρεση**: βρέφη < 6 μηνών μπορεί να αιμορραγήσουν επαρκώς ενδοκρανιακά ώστε να προκαλέσουν υποογκαιμική καταπληξία

### ΜΑΠ (Μέση Αρτηριακή Πίεση)
ΜΑΠ = ΔΑΠ + ⅓ (ΣΑΠ − ΔΑΠ). Φυσιολογική 70–100 mm Hg.

### Δείκτης Καταπληξίας (ΔΚ = Καρδιακή Συχνότητα ÷ ΣΑΠ)
- < 0,7 = φυσιολογικό
- ≥ 0,9 = υψηλότερη θνητότητα, κίνδυνος κρίσιμης αιμορραγίας
- > 1,0 = Τάξη III αιμορραγίας ή χειρότερο

### Χρώμα Δέρματος
- **Ωχρό/μαρμαρυγώδες**: ανεπαρκής οξυγόνωση περιφέρειας
- Σε **ασθενείς με σκούρα χρώση δέρματος**: αξιολόγηση χειλιών, ούλων, νυχόκρεατων, παλαμών

### Θερμοκρασία Δέρματος
- Ψυχρό δέρμα = αγγειοσύσπαση, μειωμένη δερματική αιμάτωση = **καταπληξία**
- Θερμό, ξηρό δέρμα = νευρογενής υπόταση (βλ. [[neurogenic-hypotension]])

### Χρόνος Επαναπλήρωσης Τριχοειδών
- Δοκιμή **αιμάτωσης τριχοειδικής κοίτης**, όχι δοκιμή καταπληξίας αυτή καθεαυτή
- Χρήσιμη σε συνδυασμό με άλλα σημεία
- Βελτίωση χρόνου επαναπλήρωσης = η ανάνηψη προχωρά θετικά
- Επηρεάζεται από: ψυχρό περιβάλλον, κατάγματα, υποθερμία, αρτηριοσκλήρυνση

## Παράγοντες Σύγχυσης

### Ηλικία
- **Ηλικιωμένοι**: μειωμένη αντιρροπητική ικανότητα — ήπια κάκωση μπορεί να προκαλέσει μη αντιρροπούμενη καταπληξία
- **Παιδιά/νέοι ενήλικες**: ισχυροί αντιρροπητές· μπορεί να φαίνονται καλά και να καταρρεύσουν ξαφνικά· η μη αντιρροπούμενη καταπληξία σε παιδί = **οξεία έκτακτη ανάγκη**

### Αθλητική Κατάσταση
- Αθλητές έχουν ηρεμία ΚΣ 40–50 σφ/λεπτό — ταχυκαρδία 100 σφ/λεπτό μπορεί να υποδηλώνει μαζική αιμορραγία

### Εγκυμοσύνη
- Αύξηση αίματος 45–50% — δεν δείχνει σημεία καταπληξίας έως > 30–35% απώλεια
- Το έμβρυο επηρεάζεται **πριν** εμφανιστούν σημεία υποαιμάτωσης στη μητέρα
- Τρίτο τρίμηνο: μήτρα συμπιέζει κάτω κοίλη φλέβα → αριστερή μετατόπιση μήτρας για αποκατάσταση προφορτίου

### Φάρμακα
- **Β-αναστολείς, ανταγωνιστές ασβεστίου**: αποτρέπουν αντιρροπητική ταχυκαρδία
- **ΜΣΑΦ**: εκμηδενίζουν αιμοστατική λειτουργία αιμοπεταλίων → αυξημένη αιμορραγία
- **Αντιπηκτικά**: μπορεί να αποτρέψουν πήξη για ημέρες· επηρεάζουν επιλογή τραυματολογικού κέντρου προορισμού

## Υποψία Εσωτερικής Αιμορραγίας

Όταν δεν υπάρχει εξωτερική αιμορραγία, υποψία εσωτερικής πηγής:
- **Θώρακας**: μειωμένα αναπνευστικά ψιθυρίσματα, μηχανισμός αμβλέος θωρακικού τραύματος
- **Κοιλία/λεκάνη**: εκχύμωση, διάταση, ευαισθησία, αστάθεια πυέλου, ανισότητα μηκών κάτω άκρων, εκχύμωση περινέου
- **eFAST**: προνοσοκομειακό υπερηχογράφημα για ενδοπεριτοναϊκό υγρό ή πνευμοθώρακα (όπου διαθέσιμο)

Μη αιμορραγικές αιτίες καταπληξίας (εάν αποκλειστεί αιμορραγία):
- **Καρδιακός επιπωματισμός**: διογκωμένες αυχενικές φλέβες, κωφά καρδιακά τόνοι, αμβλύ/διατιτραίνον θωρακικό τραύμα
- **Υπό τάση πνευμοθώρακας**: μειωμένα αναπνευστικά ψιθυρίσματα ετερόπλευρα, υποδόριο εμφύσημα, αναπνευστική δυσχέρεια, απόκλιση τραχείας (όψιμη εύρηση)
- **Νευρογενής υπόταση**: βραδυκαρδία, θερμά άκρα, σημεία νωτιαίας κάκωσης

## Αντιρροπούμενη vs. Μη Αντιρροπούμενη Υποογκαιμική Καταπληξία

| Ζωτικό Σημείο | Αντιρροπούμενη | Μη Αντιρροπούμενη |
|--------------|----------------|-------------------|
| Σφυγμός | Ταχυκαρδία | Έντονη ταχυκαρδία → βραδυκαρδία (προτελικό) |
| Δέρμα | Ωχρό, ψυχρό, υγρό | Ωχρό, κρύο, κηρώδες |
| Αρτηριακή πίεση | **Φυσιολογική** | Μειωμένη |
| Συνείδηση | Διατηρημένη | Αποπροσανατολισμός → κώμα |
